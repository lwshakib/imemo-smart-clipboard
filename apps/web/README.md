# iMemo Web

This is the web landing page and documentation site for iMemo Smart Clipboard, built with Next.js and TailwindCSS.

## 🚀 Development

### Run in Development

To start the web app in development mode with Turbopack:

```bash
pnpm dev
```

### Build for Production

To build the web app for production:

```bash
pnpm build
```

To start the production server:

```bash
pnpm start
```

## 🛠️ Tech Stack

- **Next.js**: React framework for production.
- **TailwindCSS**: Utility-first CSS framework.
- **shadcn/ui**: Re-usable components built using Radix UI and Tailwind CSS.
- **Turbopack**: Incremental bundler optimized for Next.js.

## 📁 Structure

- `app/`: Next.js App Router pages and layouts.
- `components/`: React components.
- `public/`: Static assets.
- `lib/`: Utility functions and shared logic.
